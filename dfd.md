## 課表與成績管理系統 - DFD 圖0

```mermaid
flowchart LR
    %% 外部實體
    User([使用者])
    
    %% 系統邏輯邊界
    subgraph System[課表與成績管理系統]
        P1[處理程序 1：課表管理]
        P2[處理程序 2：成績管理]
        P3[處理程序 3：上課提醒]
    end

    %% 資料儲存區
    D1[(課程資料庫)]
    D2[(成績資料庫)]
    D3[(提醒設定資料庫)]

    %% 流程：課表管理
    User -->|新增/編輯/刪除課程| P1
    P1 -->|更新課程資料| D1
    D1 -->|課表查詢結果| User

    %% 流程：成績管理
    User -->|輸入成績資料| P2
    P2 -->|儲存成績與平均| D2
    D2 -->|查詢成績結果| User

    %% 流程：上課提醒
    P1 -->|課程時間| P3
    User -->|設定提醒時間| P3
    P3 -->|發送提醒通知| User
    P3 -->|更新提醒設定| D3
